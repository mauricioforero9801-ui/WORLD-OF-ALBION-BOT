Política de Privacidad

Última actualización: 2025

1. Datos que se Recopilan

El bot/servidor puede recopilar datos básicos necesarios para funcionar, como:

  • ID de usuario
  • ID de servidor
  • Nombres de usuario
  • Comandos utilizados

2. Uso de los Datos

Los datos se usan únicamente para operar funciones internas del bot o el servidor.

3. Almacenamiento y Seguridad

Los datos pueden almacenarse de manera simple para funcionalidades del bot. No se comparten con terceros.

4. Eliminación de Datos

Cualquier usuario puede solicitar la eliminación de datos contactando al administrador.

5. Exoneración

El propietario no se hace responsable de accesos no autorizados, mal uso del bot o comportamientos de usuarios.

6. Modificaciones

Continuar usando el Servicio implica aceptar las actualizaciones de esta política.
